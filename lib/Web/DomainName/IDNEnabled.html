NAME

Web::DomainName::IDNEnabled - List of IDN-enabled TLDs

SYNOPSIS

  use Web::DomainName::IDNEnabled;
  ok $Web::DomainName::IDNEnabled::TLDs->{jp};
  ng $Web::DomainName::IDNEnabled::TLDs->{fr};

DESCRIPTION

The Web::DomainName::IDNEnabled module provides a list of IDN-enabled TLDs. It contains TLDs listed in the IDN-enabled TLDs (as of 2011-05-05) maintained by the Mozilla project.

VARIABLE

The module defines a variable: $Web::DomainName::IDNEnabled::TLDs. It is a hash reference. Keys of the hash are IDN-enabled TLDs in lowercase and is encoded in Punycode if necessary. Values for existing keys are always true.

SEE ALSO

IDN-enabled TLDs <http://www.mozilla.org/projects/security/tld-idn-policy-list.html>.

<http://mxr.mozilla.org/mozilla-central/source/modules/libpref/src/init/all.js>. (Search for "network.IDN.whitelist.".)
